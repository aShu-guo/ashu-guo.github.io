> useEffect 使用指南 🧭

为纯函数添加副作用，又称为"副作用函数"（纯函数内部不能有状态）

> 用法

```jsx
export default () => {
    useEffect(() => {
        //  为组件添加副作用
    })
}
```

1.参数

- 第一个参数：callback，回调函数
- 第二个参数：[]  存放依赖，当依赖发生改变时，触发回调。
  - 不传值时：每次重新渲染都会触发
  - 依赖为空数组时：副作用函数只会在组件渲染时执行一次
  - 传入值时：被传入的依赖发生改变时，会触发逻辑执行

2.执行时间

- 组件重新渲染时
- 依赖发生改变时
