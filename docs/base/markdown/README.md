# markdown 语法

## 标题

```markdown
x级标题：#*x
一级标题：# 标题
二级标题：## 标题
```

## 加粗

**Helloworld**

```markdown
**Helloworld**

```

## 删除线

~~删除线~~

```markdown
~~删除线~~
```

## 无序列表

- 行1
- 行2
- 行3

```markdown
- 行1
- 行2
- 行3
```

## 有序列表

1. 行1
2. 行2
3. 行3

```markdown
1. 行1
2. 行2
3. 行3
```

## 引入代码块

```js
console.log('hello world')
```
